The BestModel.ipynb contains all the necessary cells and code for training and making ensembled predictions on the PCL dataset.

Training was originally run on Colab with A100, so this notebook is setup for that environment.

The ensemble code was written for local use, modifications to the model paths is required to make it run. You would also need to run " **Pre-trained DeRoberta Model** " cell to make sure the model classes are avaialble for loading the models.

You can find all the models used here: https://drive.google.com/drive/folders/1xFt-oeQLGU5wKF1QSeWrYZikL0Z642WQ?usp=drive_link

The .ipynb was cleaned up slightly before submission, I did not double check if it still runs without error, but the logic and implementations are unaffected and as described in my report.
